Why docker?

==> Docker makes it really easy to install and run software without worrying about setup or dependecies

What is docker?

==> Docker is platform or ecosystem around creating and running containers

What is Container?

==> Instance of an image. Runs a program
==>A container is a standard unit of software that packages up code and all its dependencies so the application runs quickly and reliably from one computing environment to another

What is Image?

==> Single file with all dependecies and configuration required to run a program.
